# Gyaru RPG - Pixel Edition 👘✨

Un RPG de pixel art estilo Gyaru para móvil desarrollado en Unity. Explora un mundo vibrante, colecciona personajes, ¡y vence enemigos con estilo!

## 🎮 Características

- **Movimiento en Grid**: Sistema de movimiento táctico en tiles
- **Sistema de Combate RPG**: Combates por turnos contra enemigos
- **Personajes Coleccionables**: Reclutas personajes con estilo Gyaru
- **Exploración de Mapas**: Múltiples escenarios pixelados
- **UI Responsiva**: Diseño optimizado para móvil
- **Sistema de Equipos**: Gestiona tu equipo de heroínas

## 🚀 Inicio Rápido

1. Clona este repositorio
2. Abre en Unity 2022.3+
3. Abre la escena `MainMenu`
4. ¡Dale play!
